# Voice-Controlled Humanoid Serving Robot

This project is an Arduino Uno-based humanoid robot controlled through voice commands using Bluetooth technology. It can move, serve, and perform interactive robotic functions.

## Features
- Voice-controlled movement
- Forward, backward, left, right control
- Bluetooth communication
- Serving assistance
- Interactive actions

## Components Used
- Arduino Uno
- Bluetooth Module
- Motor Driver
- DC Motors
- Servo Motors
- Battery
- Chassis
